## 1. JVA（Japan's Venture Academy）とは

**JVA（Japan's Venture Academy）** は、渋谷区スタートアップ支援プログラム（Shibuya Startup Support）の公式支援を受けた、**学生主導のスタートアップ支援コミュニティ**です。

### 活動の3本柱

| 活動 | 内容 |
|------|------|
| **Bootcamp** | スタートアップ・起業家精神を学ぶ集中プログラム |
| **ネットワーキングイベント** | 学生・起業家・業界関係者をつなぐ交流イベント |
| **IB（Internship Board）** | グローバル学生とスタートアップをつなぐインターンシップマッチングプラットフォーム |

---

## 2. IB（Internship Board）とは

**IB（Internship Board）** は、JVAが運営する**インターンシップマッチングプラットフォーム**です。

### 特徴

- **グローバル人材に特化**：英語力の高い海外・留学生を主なターゲットとしており、日本語レベルに関わらず参加できるポジションが多い
- **スタートアップ向け**：採用リソースが限られるアーリー〜シード期のスタートアップが、優秀なグローバル学生インターンを採用できる
- **無料掲載**：企業はポジションを無料で掲載できる（現在）
- **LINE運用**：学生とのコミュニケーションはLINEグループを通じて行われ、求人情報の配信・応募受付を実施

### 対象学生のプロフィール

| 属性 | 傾向 |
|------|------|
| 英語力 | 80%以上がビジネス〜ネイティブレベル |
| 日本語力 | 約80%がN3〜N5（初級〜中級） |
| 専攻 | 約70%がビジネス・経済・リベラルアーツ系 |
| テック系 | 5〜8名程度（AI/ML・プログラミングスキルあり） |
| 在籍大学（例） | 東京大学・慶應義塾・早稲田・東京工業大学 など |

## 3. 学生向け：参加の流れ

```
① LINEグループに参加
        ↓
② 自動送信される挨拶メッセージ・求人情報を受け取る
        ↓
③ 興味のあるポジションのJob Cardを確認
        ↓
④ Googleフォームから応募
        ↓
⑤ 運営がCVを企業へ転送
        ↓
⑥ 企業による書類選考・面談
        ↓
⑦ マッチング（内定）
```

### 応募時に必要なもの

- 英文または日文のCV（履歴書）
- 希望ポジション
- 自己紹介・志望動機（フォーム内で入力）

### 注意事項

- 英語力・日本語力の要件はポジションによって異なる
- 週あたりの稼働時間はポジションに応じて15〜40時間と幅がある
- 応募後の選考スケジュールは各企業が管理する

## 4. 企業向け：掲載の流れ

```
① JVA運営への問い合わせ・掲載依頼
        ↓
② ポジション詳細のすり合わせ（要件・稼働時間・報酬など）
        ↓
③ Job Card（求人票）の作成・確認
        ↓
④ LINEグループでの求人配信スタート
        ↓
⑤ 応募者のCV・プロフィールを運営経由で受け取る
        ↓
⑥ 書類選考・面談を実施
        ↓
⑦ マッチング・採用決定
```

### 掲載要件の目安（マッチング率が高いポジション）

| 項目 | 推奨範囲 |
|------|---------|
| 週稼働時間 | 15〜25時間 |
| 日本語要件 | N3以下（できれば不要） |
| 英語要件 | ビジネスレベル（ネイティブ不問） |
| 経験要件 | 未経験歓迎〜中級 |
| 報酬 | ¥1,000〜¥2,000/h 程度 |

> 週35〜40時間・ネイティブ英語必須などの高難度要件はマッチング率が著しく下がるため、要件緩和を推奨しています。

## 6. オペレーション概要

IBは現在 **2名体制** で運営されています。主な業務は以下の通りです。

### 日常業務フロー

```
【学生側】
LINEグループへの参加受付
  → 自動挨拶メッセージ送信（GAS連携）
  → 求人情報・Job Cardの定期配信
  → 応募フォームの管理
  → 応募内容のスプレッドシート記録
  → 企業へのCV転送（GAS自動メール）

【企業側】
新規企業へのアウトリーチ
  → 掲載合意・Job Card作成
  → 応募者CVの送付
  → 選考状況のフォローアップ
  → マッチング（内定）確認
```

### 使用ツール

| ツール | 用途 |
|--------|------|
| **LINE** | 学生コミュニティ運営・求人配信・応募受付 |
| **Googleフォーム** | 学生の応募受付 |
| **Googleスプレッドシート** | 学生DB・企業DB・応募管理・KPI管理 |
| **Google Apps Script (GAS)** | 応募受付時の自動メール送信・データ同期 |
| **LinkedIn** | 新規学生の集客（投稿経由でLINE誘導） |

### 自動化されている業務

- **応募時の自動メール送信**：学生が応募フォームを送信すると、GASが自動で企業担当者へCVを添付してメール送信
- **KPIスプレッドシート連携**：週次KPI（LINE登録数・応募数・マッチング数など）をCLIツール経由でスプレッドシートに記録

## 7. 現状の数値（2026年3月時点）

### 主要KPI（〜2026年W09時点）

| 指標 | 累計実績 | 月次目標 |
|------|---------|---------|
| LINE新規登録数 | **46名**（W09週単位+14名） | 16.7名/月 |
| 応募数 | **15件** | 8.3件/月 |
| マッチング数 | **0件** ← 最重要課題 | 5件/月 |
| 掲載Job Card数 | **約16件** | 10件/月 |

### コンバージョン率

```
LINE登録（46名）→ 応募（15件）＝ 32.6%
応募（15件）→ マッチング（0件）＝ 0%  ← ここが詰まっている
```

### 主な課題

1. **マッチング数がゼロ**：応募者と企業要件の構造的ミスマッチが主因
   - 応募が特定企業（Jurin AI）に集中しているが、Jurin AIの要件が学生レベルを超えている
   - 企業側の選考が能動的に進んでいないケースがある
2. **企業要件のハードルが高い**：週35〜40時間・ネイティブ英語要件は学生インターンに非現実的
3. **応募前スクリーニングがない**：ミスマッチのまま応募が発生している

### 好材料

- LINE登録は月次目標を大幅超過（目標比274%）
- LinkedInへの投稿1回で15〜20名の新規登録が発生（効果的な集客チャネル確認済み）
- テック系の優秀な学生が複数在籍（東大・慶應・東工大）
