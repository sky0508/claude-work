# Assets フォルダ — JBA ブランド素材

このフォルダに置いた画像・ロゴを `shared.js` が自動で読み込みます。
ファイルが存在しない場合はデフォルトデザイン（カラーのみ）にフォールバックします。

---

## 置くべきファイル一覧

| ファイル名 | 用途 | サイズ | 形式 | 優先度 |
|-----------|------|--------|------|--------|
| `logo.png` | 全スライド右下または表紙のロゴ | 任意（幅300px推奨） | PNG（透過） | ⭐⭐⭐ 最優先 |
| `cover-bg.png` | 表紙スライドの背景 | 2540 × 1429px（16:9） | PNG | ⭐⭐⭐ 最優先 |
| `section-bg.png` | セクション区切りスライドの背景 | 2540 × 1429px | PNG | ⭐⭐ |
| `ending-bg.png` | 最終スライドの背景 | 2540 × 1429px | PNG | ⭐⭐ |
| `watermark.png` | 透かし（任意） | 任意 | PNG（透過） | ⭐ |

---

## 背景画像の作り方（Canva / Figma / PowerPoint）

- サイズ: **10" × 5.625"**（= 2540 × 1429 px 相当、16:9）
- 形式: PNG（透過不要）
- テキストを載せる領域（タイトルエリア）は**暗くしすぎない**
- ロゴは画像に焼き込まず、`logo.png` として別ファイルで用意する

---

## カラー・フォントの変更

`shared.js` の以下2箇所を編集するだけでブランドが変わります：

```js
// ブランドカラー（Hex、先頭の # は不要）
brandSkyBlue: "29B6F6",   // ← JBA のメインカラーに変更
brandBlue:    "1565C0",   // ← JBA のサブカラーに変更
headerDark:   "263238",   // ← 表紙・ヘッダーの背景色

// フォント（Google Fonts 収録のみ）
heading: "Noto Sans JP",  // ← 見出しフォント
body:    "Noto Sans JP",  // ← 本文フォント
accent:  "Montserrat",    // ← 数字・英字の強調フォント
```
