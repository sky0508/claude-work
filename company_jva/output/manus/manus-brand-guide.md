# JVA スライド作成ブランドガイド — Manus用

> このドキュメントをManusに渡すことで、JVAブランドに沿ったスライドを安定的に生成できます。
> 最終更新: 2026-03-13

---

## 1. 組織・コンテキスト

### JVAとは
- **JVA（Japan's Venture Academy）** — 渋谷区公認・学生主導のスタートアップ支援コミュニティ
- **主力事業: IB（Internship Board）** — グローバル学生×スタートアップのインターンシップマッチングプラットフォーム
- **体制**: 2名運営（小規模・アジャイル）
- **スライドの主な用途**: IB週次ミーティング資料・事業説明資料・KPI報告・外部パートナー向け提案

### スライドの典型的な読者
| 場面 | 対象読者 | 言語 |
|------|---------|------|
| IB週次ミーティング | IBチームメンバー（学生・運営） | 英語 |
| 事業紹介 | 外部スタートアップ・パートナー | 英語 |
| 社内KPI報告 | JVA経営陣 | 日本語または英語 |

## 4. スライドサイズ・レイアウト基本ルール

- **サイズ**: 16:9 / 幅10" × 高さ5.625"
- **セーフエリア（コンテンツを置いていい範囲）**: x=0.2〜9.6"、y=1.15〜5.05"
- **ヘッダーバー**: y=0〜0.46"（`#1A0508`）、その下に細いレッドライン（y=0.44〜0.49"、`#BE1229`）
- **フッターエリア**: y=5.1"以下（ロゴ・ページ番号のみ）

### スライドの構成パターン（全スライド共通）
```
[ヘッダーバー: ダーク #1A0508]
[レッドライン: #BE1229, 細線]
[コンテンツエリア: 背景 #FAF5F5]
[フッター: ロゴ右下 + ページ番号]
```

## 5. スライドタイプ別デザイン仕様

### Cover（表紙）
- **背景**: ダーク（`#1A0508`）
- **タイトル**: Montserrat Bold / 38px / 白
- **サブタイトル**: Lato / 18px / `#F5C0C8`（薄ピンク）
- **キャッチコピー**: Lato / 13px / `#C4A0A0`
- **下部アクセントライン**: `#BE1229`（水平ライン）
- **ロゴ**: 右上配置

### Section（セクション区切り）
- **背景**: `#BE1229`（レッド）
- **タイトル**: Montserrat Bold / 32px / 白 / 中央揃え
- **セクション番号バッジ**: `#E84060`の丸み矩形

### Content（コンテンツスライド） ← 最もよく使う
- **ヘッダー**: `#1A0508` + レッドライン（左にセクション名・右にページ番号）
- **背景**: `#FAF5F5`（オフホワイト）
- **タイトル**: Montserrat Bold / 大きめ / `#1A0508`
- **本文**: Lato / 13〜14px / 左揃え

### Ending（終了スライド）
- **背景**: ダーク（`#1A0508`）or カバーと同じ背景画像
- **メッセージ**: Montserrat Bold / 40px / 白 / 中央
- **連絡先**: Lato / 14px / `#F5C0C8`

## 6. コンテンツ・スタイルの好み（重要）

### やってほしいこと ✅
- **結論から先に書く**（「結論 → 根拠 → 打ち手」の順）
- **表・箇条書き・数値を多用**する（長文テキスト禁止）
- **KPI数値は大きく目立たせる**（数値が主役のスライドを作る）
- **シンプル・ミニマル**なレイアウト（1スライド1メッセージ）
- 各スライドに**明確な「So What?（だから何？）」**を持たせる
- チャートや図表を使って**データを視覚化する**
- **スピーカーノートを必ず追加**（何を話すべきかを英語で記載）

### やってほしくないこと ❌
- スライド1枚に情報を詰め込みすぎる（箇条書き6点以上は分割）
- 過度な装飾・グラデーション・シャドウ
- 文章をそのまま貼り付ける（必ず箇条書きに変換）
- ブランドカラー以外のカラーを勝手に追加する
- フォントを3種類以上使う
- アニメーション・トランジション（PPTXでは使わない）

## 7. 典型的なスライド構成（IB週次ミーティング）

> 毎週使う定番の構成。これをベースにして内容を入れ替えるだけでOK。

| # | スライドタイプ | タイトル例 | 内容 |
|---|-------------|---------|------|
| 1 | Cover | IB Weekly Meeting | 日付・テーマ |
| 2 | Content（KPI 3列） | Where Are We? | LINE登録数・応募数・マッチング数 |
| 3 | Content（Before/After） | What's the Problem? | 想定 vs 実態の比較 |
| 4 | Content（課題放射） | Root Cause Analysis | 問題の原因を放射状に整理 |
| 5 | Content（比較表） | Options & Trade-offs | 対応策A vs 対応策B |
| 6 | Content（番号付きカード） | Today's Goals | ミーティングで決めること3点 |
| 7 | Ending | Let's Decide. | 連絡先・日付 |

**目安枚数**: 7〜10枚
**言語**: 英語（IBチーム用）

## 8. KPIデータの表記ルール

IB事業の主要KPIは以下の通り。スライドに数値を載せる場合はこのフォーマットで：

| KPI | 表記例 |
|-----|-------|
| LINE登録数（累計） | `59 LINE Registrations` / `+10 this week` |
| 応募数（累計） | `23 Applications` / `+2 this week` |
| マッチング数 | `0 Matches` ← 最重要課題として強調 |
| 掲載Job Card数 | `16 Active Positions` |

**カラーコード（KPI状態）**:
- 良好・目標超過: `#27AE60`（グリーン）
- 注意・停滞: `#F59E0B`（イエロー）
- 問題・ゼロ: `#BE1229`（レッド）

## 10. JVA事業コンテキスト（背景知識として渡す）

Manusがコンテンツを生成する際に参考にしてください。

```
JVA IB（Internship Board）の現状（2026年3月時点）:

プラットフォーム概要:
- グローバル学生（英語力高・日本語初級）とスタートアップをつなぐインターンマッチングサービス
- LINEグループ経由で学生とコミュニケーション
- 無料掲載。GAS（Google Apps Script）で応募時の自動メール送信を自動化済み

主要KPI（Week 09時点）:
- LINE登録: 59名（目標比 +177%）
- 応募数: 23件（累計）
- マッチング数: 0件 ← 最大ボトルネック
- 掲載ポジション: 16件 / 約7社

主な課題:
- マッチング数がゼロ（最重要課題）
- 原因①: 企業側の選考が進まない（応答なし・SLAなし）
- 原因②: 学生と企業要件のミスマッチ（週35-40h・ネイティブ英語要件は非現実的）
- 原因③: 応募前のスクリーニングがない

好材料:
- LINE登録は目標の274%超過
- LinkedIn投稿1回で15-20名の新規登録
- テック系優秀学生（東大・慶應・東工大）が複数在籍
```
