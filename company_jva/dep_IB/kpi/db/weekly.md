## 週の定義

## カラム定義

| カラム | 説明 |
|--------|------|
| 週 | JBA週番号（W01 = 1/13〜） |
| LINE新規 | 週の純増実ユーザー数 |
| LINE累計 | 週末（水曜）時点の累計実ユーザー数 |
| Application | その週の新規応募件数 |
| Match | その週の新規マッチング件数 |
| JobCard | その週の新規求人票作成数 |
