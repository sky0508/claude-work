# KPI Database — Conversion Rates

> フロー全体の転換率を週次・累計で追跡。`/weekly_kpi` 実行時に自動更新。

## フロー

```
LINE登録 → Application → 書類選考 → 面接 → Match
```

| 指標 | 計算式 |
|------|--------|
| LINE→App率 | Application累計 ÷ LINE累計 × 100 |
| App→Match率 | Match累計 ÷ Application累計 × 100 |
