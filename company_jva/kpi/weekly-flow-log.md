# Weekly Flow Log（週次フロー実績）

> 実績はここに**週単位で記録**する。スプレッドシート「01_Weekly_Flow_Log」と対応。  
> 入力された数値をもとに、コンバージョン率・月次サマリーがスプレッドシート側で自動計算される。  
> **LINE累計登録は、このシートにある数値のみを合計して計算する**（他ソースと合算しない）。

## 運用ルール

- 1行 = 1週分の実績。週の開始日（月曜）または週末日で識別。
- 数値を修正したら、必要に応じてスプレッドシートへ反映（POST `appendRow` / `updateCell` / `updateRange`）。  
  反映方法は `kpi/README.md` の「KPIを修正する」を参照。
- 列はスプレッドシート「01_Weekly_Flow_Log」のヘッダーに合わせる。

## スプレッドシートへの反映

- **新規週の追加**: GAS の `appendRow` で 01_Weekly_Flow_Log に1行追加。
- **既存セルの修正**: `updateCell` または `updateRange` で該当セルを更新。  
  token と URL は `config/sheets.json` の `sheets.kpi` を参照。
