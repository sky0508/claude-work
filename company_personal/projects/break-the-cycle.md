# Break the Cycle — エネルギー回復プロジェクト

> 秘書・コーチとして一緒に取り組む個人プロジェクト。
> 「やりたいことが多すぎて動けない → 動画回避 → 自己嫌悪」のループを断ち切る。

*開始: 2026-03-13*

---

## 現状の診断

**症状:**
- エネルギー・モチベーション低下（ここ数日）
- やりたいことはたくさんある、でもどれからやるべきか分からない
- めんどくさくてYouTube・ショート動画に逃げてしまう
- 何もできないまま1日が終わる → 自己嫌悪

**根本構造:**
```
選択肢が多すぎる
    ↓
どれから？が分からない（選択麻痺）
    ↓
不安・重さを感じる
    ↓
動画で回避（手軽な快楽）
    ↓
時間が溶ける → 自己嫌悪
    ↓
さらに選べなくなる（ループ）
```

**性質:** 意志力の問題ではない。「頭の中の処理オーバーロード」が原因。

---

## アプローチ方針

1. **頭の中を外に出す** — やりたいことを全部吐き出して、頭を軽くする
2. **今日1つだけ決める** — 完璧な優先順位より「今日これだけ」が先
3. **動画の代替行動を仕込む** — 回避したくなったときの逃げ道を変える
4. **小さな勝ちを積む** — 自己嫌悪を自己信頼に変えていく
