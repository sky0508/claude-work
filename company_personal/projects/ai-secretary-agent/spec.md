> 作成日: 2026-03-15

---

## コンセプト

**"どこからでも使える、自分専用の秘書"**

今Cursorで動いている秘書スキル（タスク管理・カレンダー・メール対応）を、
**Webアプリとして独立させる**プロジェクト。

- Cursor（エディタ）を開かなくてもスマホやブラウザから使える
- ツール（Gmail / Calendar / Notion）と直接つながっていてリアルタイムに動く
- AIが「優先すべきタスク」「返信すべきメール」「入れられるミーティング」を自動で判断して提示する

## 想定する1日の使い方

### 朝（Morning Briefing）
1. アプリを開く
2. **今日の予定**（Google Calendar）と**未返信メール数**（Gmail）がダッシュボードに表示される
3. AIが「今日のMustタスク」を3件ピックアップして提示
4. タスクマトリックス全体を確認して優先度を調整

### 日中（Daytime）
- **タスク追加**: テキストで入力 → AIが象限を自動判定してマトリックスに配置
- **タスク完了**: 1タップでチェック→完了欄に移動
- **日程調整**: 相手の名前と目的を入力 → カレンダーから空き時間を算出して候補3件を提示
- **メール対応**: 未返信リストから選択 → AIが返信ドラフトを生成 → 確認後送信

### 夜（Evening Check）
- 今日の完了タスクを振り返り
- 明日以降に持ち越すタスクを確認
- Notionに進捗を自動同期

## 主要機能

### 1. タスクマトリックス（メイン画面）

```
             緊急      非緊急
重要   │ Q1 (今すぐ) │ Q2 (計画的に) │
非重要 │ Q3 (委任)   │ Q4 (後回し)  │
```

- ドラッグ&ドロップで象限を変更
- タスクカードには：タイトル / 締め切り / 関連プロジェクト / ステータス
- AIが新規タスクの象限を自動提案
- フィルター: JVA業務 / 個人 / 両方

### 2. カレンダー連携（Google Calendar）

- 今日・今週の予定をダッシュボードに表示
- 日程調整モード：
  - 入力: 相手名・目的・所要時間
  - 出力: 今後7日間の空き時間から候補3件
  - オプション: 候補をそのままコピーできる文面も生成

### 3. メール対応（Gmail）

- 未返信・要対応メールを自動抽出（条件: 受信2日以上 / 重要ラベル / プロモーション除外）
- 各メールに対して：
  - 一言サマリー（何を求めているか）
  - 推奨アクション（返信する / 後回し / 対応不要）
- 返信ドラフト生成 → 確認後に送信（**送信は必ず確認を取る**）

### 4. Notion同期

- タスクの追加・完了・象限変更をNotionのDBに双方向同期
- セッション終了時に進捗サマリーをNotionに書き込み

### 5. AIチャット（サイドパネル）

- 何でも相談できるチャットUI
- 「〇〇のメールに返信して」「来週火曜に〇〇と1時間入れて」などの自然言語命令に対応
- 実行前に確認を取るアクション: メール送信 / カレンダー追加

## 非機能要件・制約

- **確認ルール**: メール送信・カレンダー追加は必ずユーザー確認を挟む
- **データ保持**: タスクデータはNotionをマスターとして持つ
- **アクセス**: ブラウザ・スマホ両対応
- **認証**: OAuth連携済みの自分のアカウントのみ（Phase 1）
