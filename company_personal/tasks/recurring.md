# 定期タスク

> カンパニーをまたいで繰り返す定期業務。/fin_s が自動更新する。

## 毎日

- [ ] 今日のTODO確認（/company で開始時）
- [ ] Inbox の消化

## 毎週

- [ ] 週次レビュー（各カンパニーの進捗確認）
- [ ] IB: Manus週次レポート受領 → レポート貼り付け → 「今週のレポートだよ」→ 自動でKDB更新＋戦略レポート/英語スライド生成（`/weekly_strategy`）
- [ ] IB: LINE配信スケジュール確認
- [ ] IB: 応募状況・KPI確認（`./kpi/kpi.sh status` で即確認可）
- [ ] IB: KPI週次実績を記録（**毎週水曜**に `./kpi/kpi.sh record` → スプレッドシートに自動反映）
- [ ] IB: **LinkedIn投稿 1回以上**（LINE登録の主要ドライバーと判明。月2回で月次目標達成可能）
- [ ] IB: 掲載企業への選考状況フォロー（I&N・GlobalDeal等）

## 毎月

- [ ] 各カンパニーの月次振り返り
- [ ] IB: 掲載企業の状況確認（Active/Not Active）
- [ ] IB: 新規企業開拓 進捗確認

## 随時（トリガーベース）

- 新しい企業からフォーム申し込み → Notionへ登録 → sales/clients に追加
- 学生から応募 → マッチング候補確認 → hr に記録
- マッチング成立 → KPI更新 → pm/projects/ib-overall.md 更新
