# save — 作業状態をNotionに保存

## 概要

現在のセッションの作業状態をNotionのアクティブコンテキストページに保存する。
追記ではなく「## 作業状態」セクションを**全面書き直し**する（情報の混在防止）。

## 保存先の決定

1. 会話中にNotionページが渡されていれば → そのページに保存
2. 渡されていない場合 → デフォルトのresumeページに保存
   - `https://www.notion.so/Active-Context-Claude-Code-327f995fe92a81a7833be53abf1b1d29`

## 保存する内容

以下のフォーマットで「## 作業状態」セクションを書き直す：

```
## 作業状態
最終更新: YYYY-MM-DD HH:MM

### 今やっていること
（現在のタスク・目的）

### 完了済み
（このセッションで終わったこと）

### 残タスク
（まだやること・次のステップ）

### 重要なコンテキスト
（引き継ぎに必要な情報・決定事項・注意点）
```

## 手順

1. 保存先ページを確認（渡されたページ or デフォルトresumeページ）
2. MCPでそのページを取得
3. 現在の作業状態を整理
4. 「## 作業状態」セクションを新しい内容で全面置き換え
5. 「保存しました」と報告
