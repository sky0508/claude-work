# resume — Notionから作業状態を復元

## 概要

Notionに保存された作業状態を読み込み、前回の続きから作業を再開する。

## トリガー

- 「続き」
- 「続きやって」
- NotionページURLを渡して「続き」

## 手順

1. **ページ特定**
   - 会話でNotionページURLが渡されていれば → そのページを読む
   - なければ → デフォルトのresumeページを読む
     `https://www.notion.so/Active-Context-Claude-Code-327f995fe92a81a7833be53abf1b1d29`

2. **コンテキスト読み込み**
   - MCPでページ内容を取得
   - 「## 作業状態」セクションを解析

3. **状況報告**
   以下のフォーマットで現状を報告する：
   ```
   前回の続きです。

   今やっていたこと: [内容]
   完了済み: [リスト]
   残タスク: [リスト]

   「残タスク」の[最初のタスク]から始めますか？
   ```

4. ユーザーの確認を得て作業を再開
